# Step 1: Define Purpose

## Objective

Clarify what the system prompt needs to accomplish.

## Instructions

### Part A: Core Purpose

Ask:
1. "What task should this prompt guide?"
2. "Who/what is the target audience?"
3. "What output format is expected?"

### Part B: Target Model

"Which model will use this prompt?"
- GPT-4o
- Claude Sonnet
- Gemini Pro
- Other

(Model choice affects optimal structure)

### Part C: Success Criteria

"How will you know the prompt is working?"
- Specific outputs
- Quality metrics
- Edge cases to handle

## Outputs

- [ ] Purpose statement: 1-2 sentences
- [ ] Target model selected
- [ ] Success criteria defined
